cycloped.io

Cycloped.io is a framework consisting of three elements:

  • a reference taxonomy based on Cyc ontology
  • an open source system allowing for mapping and classifying external knowledge bases to the taxonomy
  • a classification of the Wikipedia articles into cycloped.io taxonomy

Taxonomy

There are three levels of the taxonomy:

  • Top-level - 41 abstract types
  • Mid-level - 1119 meaningful types
  • Low-level - 27 thousand types

Source code

The source code of the system is available at github.com/cycloped-io/cycloped-io

Classification of English Wikipedia articles

  • Fine-grained classification - classification based on low-level types.
  • Coarse-grained classification - classification based on mid-level types.

Example

http://en.wikipedia.org/wiki/Elvis_Presley:
  • Main type: Person
  • Prominent types:
    • Singer
    • MusicalPerformer
  • Roles:
    • Actor
    • ActorInMovies
    • Artist
    • Artist-Performer
    • Athlete
    • FamousHuman
    • MilitaryPerson
    • Musician
    • Singer
    • Soldier
  • Other types:
    • Baritone-Singer
    • MaleHuman
    • MusicalPerformer
    • UnitedStatesPerson
